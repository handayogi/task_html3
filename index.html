HTML (HyperText Markup Language) adalah bahasa deskriptif yang menentukan struktur halaman web. Pada tahun 1990, sebagai bagian dari visinya tentang Web, Tim Berners-Lee mendefinisikan konsep hypertext, yang diformalkan Berners-Lee tahun berikutnya melalui markup yang sebagian besar didasarkan pada SGML. IETF mulai secara resmi menetapkan HTML pada tahun 1993, dan setelah beberapa draf merilis versi 2.0 pada tahun 1995. Pada tahun 1994 Berners-Lee mendirikan W3C untuk mengembangkan Web. Pada tahun 1996, W3C mengambil alih pekerjaan HTML dan menerbitkan rekomendasi HTML 3.2 setahun kemudian. HTML 4.0 dirilis pada tahun 1999 dan menjadi standar ISO pada tahun 2000.

CSS (Cascading Style Sheets) adalah bahasa deklaratif yang mengontrol bagaimana halaman web terlihat di browser. Browser menerapkan deklarasi gaya CSS ke elemen yang dipilih untuk menampilkannya dengan benar. Deklarasi gaya berisi properti dan nilainya, yang menentukan tampilan halaman web. CSS adalah salah satu dari tiga teknologi inti Web, bersama dengan HTML dan JavaScript. CSS biasanya memberi gaya pada elemen HTML, tetapi juga dapat digunakan dengan bahasa markup lain seperti SVG atau XML. Aturan CSS adalah serangkaian properti yang dikaitkan dengan pemilih.

JavaScript (atau "JS") adalah bahasa pemrograman yang paling sering digunakan untuk skrip sisi klien dinamis pada halaman web, tetapi juga sering digunakan di sisi server, menggunakan runtime seperti Node.js. JavaScript tidak boleh disamakan dengan bahasa pemrograman Java. Meskipun "Java" dan "JavaScript" adalah merek dagang (atau merek dagang terdaftar) Oracle di AS dan negara lain, kedua bahasa pemrograman tersebut sangat berbeda dalam sintaksis, semantik, dan kasus penggunaan. JavaScript terutama digunakan di browser, yang memungkinkan pengembang untuk memanipulasi konten halaman web melalui DOM, mengambil konten dari server menggunakan fetch() API, menyimpan data kompleks menggunakan IndexedDB, menggambar grafik dengan kanvas, berinteraksi dengan perangkat yang menjalankan browser melalui berbagai API, dan banyak lagi. JavaScript adalah salah satu bahasa yang paling umum digunakan di dunia, karena pertumbuhan dan peningkatan kinerja API yang tersedia di browser baru-baru ini.
